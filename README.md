# Muscle Mantra 💪

A modern Gym & Fitness Website built using HTML. This project showcases a professional fitness website with information about workouts, trainers, membership plans, and fitness services.

## 🚀 Features

* Home Page
* About Us Section
* Membership Plans
* Professional Trainers
* Workout Programs
* BMI Calculator Section (UI)
* Success Stories
* Fitness Gallery
* Contact Us Form
* Responsive and User-Friendly Layout

## 🛠️ Technologies Used

* HTML5

## 📂 Project Structure

```text
Muscle-Mantra/
│
├── index.html
├── images/
└── README.md
```

## ▶️ How to Run

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Explore the website.
